# H1 Это проектная работа, спринт 02.

Данное веб-приложение представляет пользователям возможность 
поиска и отображения видео записей 
из заранее заданного спектра доступных видео-камер в общественных пространствах.

## H2 Основная область интерфейса: 
- видео плеер, позволяющий воспроизвести выбранное видео

## H2 Боковая панель справа: 
- список найденных видео записей, разрешённых и доступных к просмотру действующим законодательством

## H2 Панель инструментов в нижней части интерфейса: 
- текстовое поле поиска видео роликов
- фильтры поиска по времени суток, как атрибуте поиска искомого видео
- кнопка "Найти" - запуск поиска согласно указанным параметрам поиска


## H2 Обратите внимание: 
Также, в данном веб приложении реализована поддержка достуности функционала интерфейса: 
- выделение элементов управления при навигации с клавиатуры
- выделение элементов управления при навигации с использованием мыши

Также: реализованы нестандартные, заказные элементы управления "чекбокс".
